# Cronjob Plan Task

这是一个用于管理定时任务的项目，基于 Node.js 和 BullMQ 实现。

## 功能

- 创建和管理定时任务
- 记录任务执行情况
- 提供 RESTful API 接口
- 支持高可用性调度

## 技术栈

- Node.js
- Express
- BullMQ
- Redis
- MongoDB (或其它数据库)

## 安装

```bash
npm install
```

## 使用

```bash
npm start
```

## 测试

```bash
npm test
```
